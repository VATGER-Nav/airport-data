# Airport Data

Manages the airport data used on the VATSIM Germany homepage.

### Airport type:

```
[[airport]]
icao = "EDDY"

[[airport.links]]
category = "Scenery" | "Charts" | "Briefing" | "GSX Config" | NONE
name = "Display Name 1"
url = "https://example.url"
```
